#### ❗️yarn 및 의존성 패키지 설치

본 Repository 는 yarn workspace 모노레포로 구성되어있습니다. 실행전 `yarn` 을 설치해주세요

```bash
npm install --global yarn
yarn
```

#### 📜 클라이언트 실행

```bash
yarn client dev
```

📜 서버 실행

```bash
yarn server start:dev
```
